    Eat-Da-Burger:

    Eat-Da-Burger is a full stack web application that utilizes an MVC (model-view-controller) and a database to allow users to enter a burger that sounds delicious, which will append that burger to the DOM. When their new burger is appended, the user can then devour the burger with the "Devour" button, moving the burger to the opposite side of the page. They then may click the "Delete" button to completely rid the page of the burger. This utilizes Bootstrap, NodeJS, Express, and MYSQL.

    Deployed application: https://burger-homework123456.herokuapp.com/

    GitHub repository: https://github.com/kaitlinkenney/eat-the-burger

    FiveWire:

    FiveWire is an application that uses the Spotify API, npm packages sequelize and express, MySQL database, Animate.Css, and UIKit. FiveWire allows users to create an account, see a featured playlist with trending songs, search for songs, add and delete songs on a playlist, create and delete playlists, and join other users in the "jukebox" room with a live music queue. When in the jukebox room, users can add songs to the queue, and play/pause the current song.

    Deployed application: https://group5-proj2.herokuapp.com/

    GitHub repository: https://github.com/nickkdb/Group5-Proj2

    SmartTrvl:

    Travel during the COVID-19 pandemic poses extra risk. The goal of SmartTrvl is to simplify the unknown risk of travel to countries that the user is trying to travel to as well as display tickets within the user's price range. Will allow users to find plane tickets to countries within their budget. Utlilizes Materialize for styling. APIs Used: SkyScanner & COVID-19 Coronavirus Statistics.

    Deployed application: https://kaitlinkenney.github.io/SmartTrvlApp/
